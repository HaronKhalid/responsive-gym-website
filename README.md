# Prime Fitness - Static SPA

A premium, highly responsive single-page fitness website built entirely with HTML, custom CSS, and Vanilla JavaScript. 
This project showcases a modern, dark-themed gym aesthetic with dynamic animations, calculators, and scrolling features, all while maintaining zero complex build dependencies.

## Features
- **Zero Build Tools**: No React, no Webpack, no node_modules. Open `index.html` directly in your browser.
- **Tailwind CSS (via CDN)**: Rapid, utility-first styling mirroring the exact premium design system.
- **Vanilla JavaScript Logic**: 
  - Interactive BMI & Calorie calculators
  - Custom IntersectionObserver scroll reveal animations
  - Before/After transformation slider with custom CSS clip-paths
- **Mobile First & Fully Responsive**: Works perfectly on phones, tablets, and massive desktop screens.

## File Structure
- `index.html`: The core structure, layout, and content.
- `css/style.css`: Custom animations, custom scrollbars, and keyframe definitions.
- `js/main.js`: Interactivity, form calculations, and animation triggers.

## Quick Start
1. Clone this repository: `git clone https://github.com/HaronKhalid/responsive-gym-website.git`
2. Open `index.html` in your favorite web browser.
3. Enjoy!
